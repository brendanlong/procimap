Module Hierarchy

  • ProcImap: The ProcImap package fulfills two purposes:
    • ProcImap.ImapMailbox: This module provides the ImapMailbox class, which is a wrapper around the imaplib module of the standard library, and a full implementation of the mailbox.Mailbox interface.
    • ProcImap.ImapMessage: This module contains the ImapMessage class, which derives from mailbox.Message.
    • ProcImap.ImapServer: This module contains a wrapper around an IMAP Server.
    • ProcImap.Utils
      • ProcImap.Utils.Gmail: This module contains functions that make it easier to work with Gmail.
      • ProcImap.Utils.MailboxFactory: This module contains the MailboxFactory class, together with some Exceptions that are thrown by MailboxFactory, and the pathgenerator functions used for the default mailbox types (see documentation of MailboxFactory class)
      • ProcImap.Utils.Processing: This module contains general functions for processing mail locally e.g.
      • ProcImap.Utils.Server: This module contains functions that work directly on an online ImapServer/ImapMailbox.
    • ProcImap.imaplib2: Threaded IMAP4 client.
